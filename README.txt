BATTERY LEVEL MONITOR AND ALARM
-------------------------------

Generates a sound alarm and prompts the user to plug/unplug the laptop:
	- when laptop is plugged in with battery level having reached or exceeded the upper limit.
	- when laptop is unplugged with battery level at or below the lower limit.

This program can be executed automatically after each boot by adding this as a scheduled task in Task Scheduler.

Add task: <path_to_python.exe> <path_to_batteryMonitor.py>
